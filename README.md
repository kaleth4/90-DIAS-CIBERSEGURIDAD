## 📌 Introducción

Este plan de 90 días está diseñado para construir una base sólida en ciberseguridad, integrando teoría, práctica y herramientas reales del campo. Cada fase escala en complejidad y está alineada con certificaciones y roles demandados en la industria.

```
Redes → Linux → Seguridad defensiva → Python → Análisis → Hacking ético → Forense → Carrera
```

## 🌐 Fase I — Cimientos: Infraestructura y Redes
### `Días 1 - 28`

> Modelos OSI/TCP-IP, seguridad en protocolos y administración de servidores Linux.

<details>
<summary><b>📅 Días 1-14: Conceptos de Redes (Network+)</b></summary>

- 📺 Estudia la lista de reproducción **N10-009 del Profesor Messer**
- ✅ Completa preguntas de práctica por cada módulo
- 🎯 Objetivo: dominar subnetting, protocolos y topologías

**Recursos:**
- [Professor Messer N10-009](https://www.professormesser.com/network-plus/n10-009/n10-009-video/n10-009-training-course/)

</details>

<details>
<summary><b>📅 Días 15-28: Fundamentos de Linux</b></summary>

- 🐧 Explora **Linux Journey** (linuxjourney.com)
- 🎓 Completa **Linux Unhatched** en Cisco NetAcad
- 🔬 Realiza todos los laboratorios de **LabEx**
- 🎯 Objetivo: administración de servidores, permisos, scripting básico

**Recursos:**
- [Linux Journey](https://linuxjourney.com/)
- [Cisco NetAcad - Linux Unhatched](https://www.netacad.com/)
- [LabEx Linux Labs](https://labex.io/)

</details>

---

## 🔐 Fase II — Núcleo de Seguridad: Defensa y Criptografía
### `Días 29 - 63`

> IAM, algoritmos de cifrado, firewalls, IDS/IPS, Python y análisis de tráfico.

<details>
<summary><b>📅 Días 29-35: Conceptos de Seguridad (Security+)</b></summary>

- 📺 Recursos de **Profesor Messer** o **Pete Zerger** (SY0-701)
- 🎯 Objetivo: principios CIA, gestión de riesgos, controles de seguridad

**Recursos:**
- [Professor Messer SY0-701](https://www.professormesser.com/security-plus/sy0-701/sy0-701-video/sy0-701-training-course/)

</details>

<details>
<summary><b>📅 Días 36-49: Python para Seguridad</b></summary>

- 🐍 Track de Python en **Codecademy**
- 📖 Libro: *"Learn Python the Hard Way"*
- 🎯 Objetivo: automatizar tareas de seguridad, scripting ofensivo básico

**Recursos:**
- [Codecademy Python](https://www.codecademy.com/learn/learn-python-3)

</details>

<details>
<summary><b>📅 Días 50-63: Análisis de Tráfico, IDS/IPS y Git</b></summary>

- 🦈 Aprende **Wireshark** y **TCPdump** para inspección de paquetes
- 🚨 Configura **Suricata** para detección de intrusiones
- 🔀 Control de versiones con **Git Immersion** o Try Git
- 🎯 Objetivo: detectar tráfico malicioso, análisis de malware en red

**Recursos:**
- [Wireshark Docs](https://www.wireshark.org/docs/)
- [Suricata](https://suricata.io/)
- [Git Immersion](https://gitimmersion.com/)

</details>

---

## 💻 Fase III — Ofensiva: Hacking Ético
### `Días 64 - 84`

> SIEM, análisis de logs, seguridad en la nube y pentesting en entornos controlados.

<details>
<summary><b>📅 Días 64-70: SIEM y Análisis de Logs (ELK Stack)</b></summary>

- 📊 Aprende **Elasticsearch, Logstash, Kibana**
- 🖥️ Laboratorios en máquinas **Ubuntu y Windows**
- 🎯 Objetivo: correlación de eventos, detección de anomalías

</details>

<details>
<summary><b>📅 Días 71-77: Seguridad en la Nube</b></summary>

- ☁️ Explora seguridad en **AWS**, **Azure** o **GCP**
- 🎮 Practica con **AWS Cloud Quest** (gamificado)
- 🎯 Objetivo: IAM en cloud, misconfiguraciones comunes, S3 buckets

**Recursos:**
- [AWS Cloud Quest](https://aws.amazon.com/training/digital/aws-cloud-quest/)

</details>

<details>
<summary><b>📅 Días 78-84: Pentesting y Explotación</b></summary>

- 💣 Practica en **Hack The Box** y **VulnHub**
- 🔎 Investigación con **Splunk** en entornos Windows/Ubuntu
- 🎯 Objetivo: explotar vulnerabilidades reales en entorno controlado

**Recursos:**
- [Hack The Box](https://www.hackthebox.com/)
- [VulnHub](https://www.vulnhub.com/)

</details>

---

## 🔍 Fase IV — Respuesta y Análisis: Forense y Carrera
### `Días 85 - 95+`

> Threat hunting, forense digital y preparación para el mercado laboral.

<details>
<summary><b>📅 Días 85-90: Threat Hunting y Forense</b></summary>

- 🦅 Threat hunting con **Velociraptor** y **Osquery**
- 🔐 Investiga intentos fallidos de RDP y accesos no autorizados en Linux
- ⚙️ Automatización con **Splunk SOAR**
- 🎯 Objetivo: detectar TTPs de atacantes, respuesta ante incidentes

</details>

<details>
<summary><b>📅 Días 91-95: Preparación de Carrera</b></summary>

- 📄 Crea un CV de una página con plantillas para ciberseguridad
- 💼 Aplica a roles **SOC Analyst** / **Security Analyst** en LinkedIn e Indeed
- 🏆 Documenta tu progreso en GitHub con este repositorio

</details>
